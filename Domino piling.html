الدرس الأول: تحدي رقعة الدومينو

كيف نغطي مساحة ثنائية الأبعاد رياضياً؟

🧩 المشكلة المطروحة

تخيل أن لديك رقعة مستطيلة (مصفوفة ثنائية الأبعاد) تتكون من M عمود و N صف. بداخل هذه الرقعة مربعات صغيرة.
ولديك عدد غير محدود من قطع الدومينو. كل قطعة دومينو حجمها 2 × 1 (أي تغطي مربعين متجاورين بالضبط).

شروط المسابقة الصارمة:

  • كل قطعة يجب أن تغطي خانتين كاملتين (لا يمكن كسرها).
  • يمنع تقاطع أو تداخل القطع مع بعضها البعض.
  • يجب أن تكون جميع القطع داخل حدود الرقعة بالكامل.

الهدف: ما هو "أكبر عدد ممكن" من قطع الدومينو التي يمكننا وضعها؟

الدرس الثاني: التحليل الرياضي للحالات

الكمبيوتر لا يحتاج لرسم الرقعة، يحتاج فقط للرياضيات!

عدد المربعات الكلي في الرقعة هو المساحة (N * M). وبما أن كل قطعة تغطي مربعين، فمنطقياً سنقسم المساحة الكلية على 2. لكن دعونا نثبت ذلك علمياً من خلال دراسة 3 حالات:

1️⃣ الحالة الأولى: عدد الصفوف (N) زوجي

إذا كان N زوجياً (N % 2 == 0)، يمكننا تغطية المصفوفة بالكامل بوضع القطع بشكل عمودي تماماً، ولن يتبقى أي فراغ.
المعادلة: عدد القطع = (N * M) / 2

2️⃣ الحالة الثانية: عدد الأعمدة (M) زوجي

بشكل مشابه، إذا كان M زوجياً (M % 2 == 0)، نضع القطع بشكل أفقي لتغطية كل شيء دون فراغات.
المعادلة: عدد القطع = (N * M) / 2

3️⃣ الحالة الثالثة: كلا العددين (N و M) فرديان

هنا التحدي! لأن ناتج المساحة (N * M) سيكون رقماً فردياً. نضع القطع بشكل عمودي حتى يتبقى صف أخير واحد (لأن N فردي). وفي هذا الصف نضعها أفقياً حتى تتبقى خانة واحدة فقط في الزاوية لا يمكن تغطيتها.

المعادلة: نطرح الخانة الفارغة ثم نقسم: (N * M - 1) / 2

الدرس الثالث: تفكيك الكود الأساسي

ترجمة الحالات الرياضية الثلاث إلى كود C++.

اضغط على علامة التعجب ! بجوار الكود لتفهم وظيفة كل سطر.

  • cin >> N >> M;
    تعريف المتغيرات واستقبال أبعاد الرقعة من نظام التحكيم في المسابقة.
  • if (N % 2 == 0)
    يختبر الحالة الأولى. إذا كان الباقي صفر (N زوجي)، يطبع النصف.
  • else if (M % 2 == 0)
    يختبر الحالة الثانية. إذا كان M زوجي، يطبع النصف أيضاً.
  • else
    إذا فشل الشرطان السابقان، فهذا يعني حتماً أن كلاهما فردي (الحالة الثالثة). فيطرح 1 من المساحة الكلية ثم يقسم على 2.
#include <iostream>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M; // إدخال أبعاد المصفوفة

    // الحالة الأولى: الصفوف زوجية
    if (N % 2 == 0) {
        cout << (N * M) / 2 << endl;
    }
    // الحالة الثانية: الأعمدة زوجية
    else if (M % 2 == 0) {
        cout << (N * M) / 2 << endl;
    }
    // الحالة الثالثة: كلاهما فردي
    else {
        cout << (N * M - 1) / 2 << endl;
    }
}

الدرس الرابع: التتبع التفاعلي (المشغل الذكي)

لنجرب تتبع كود الرقعة ذات الـ 3 صفوف و 3 أعمدة سطراً بسطر.

شاشة المراقبة

الصفوف (N)
3
الأعمدة (M)
3
المساحة (N * M)
9
النتيجة (الطباعة)
-
int main() {
  int N = 3, M = 3;
if (N % 2 == 0) {
cout << (N * M) / 2 << endl;
}
else if (M % 2 == 0) {
cout << (N * M) / 2 << endl;
}
else {
cout << (N * M - 1) / 2 << endl;
}
}

الدرس الخامس: سر المحترفين (O(1))

كيف نختصر كل هذا الكود الطويل في سطر واحد فقط؟

خدعة القسمة الصحيحة في C++ 🤯

في لغة C++، عندما نقسم رقماً فردياً صحيحاً (int) على 2، فإن اللغة تحذف الكسر العشري تلقائياً.
مثال: المعادلة 9 / 2 لن تعطي 4.5 بل ستعطيك 4 مباشرة وبدون أي تقريب!

هل لاحظت شيئاً؟ هذا يعني أننا لسنا بحاجة لطرح 1 في الحالة الثالثة أبداً! الكمبيوتر سيطرحه تلقائياً بتجاهل الكسر. وهذا يعني أن الحالات الثلاث يمكن دمجها في معادلة رياضية واحدة، وسطر كود واحد!

  • هذا هو كود المحترفين! اختصرنا الشروط الثلاثة لأن المعادلة (N * M) / 2 تعطي الناتج الصحيح دائماً بفضل طبيعة القسمة الصحيحة في لغة C++.
  • التعقيد الزمني هو O(1) لأنه ينفذ عملية حسابية واحدة فقط، بغض النظر عن حجم الرقعة حتى لو كانت مليار × مليار.
#include <iostream>
using namespace std;

int main() {
    int N, M;
    cin >> N >> M;
    
    // سطر واحد فقط يحل كل الحالات!
    cout << (N * M) / 2 << endl; 
}
