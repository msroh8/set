الدرس الأول: خزانة الملفات الذكية (Map)

فهم كيفية ربط البيانات ببعضها في لغة C++.

🗄️ التمهيد الجاذب (دفتر العناوين)

"تخيلوا أننا نريد تصميم برنامج يسجل أعمار الطلاب في المدرسة. إذا سألت الكمبيوتر: 'كم عمر الطالب أحمد؟'، أريده أن يجيبني فوراً '14 سنة'. في المصفوفات العادية، لا يمكننا استخدام أسماء الأشخاص للبحث، بل نستخدم أرقاماً (مثل 0, 1, 2). هنا يتدخل بطلنا الجديد: الـ Map (الخريطة)."

الـ Map هو عبارة عن "خزانة ذكية" تقوم بتخزين البيانات على شكل أزواج مترابطة:

  • المفتاح (Key): مثل الملصق الموجود على الدرج (مثال: اسم الطالب). يجب أن يكون فريداً ولا يتكرر.
  • القيمة (Value): هي المعلومة الموجودة داخل الدرج (مثال: عمر الطالب). يمكن أن تتكرر.
  • الترتيب التلقائي: الخزانة تقوم بترتيب الأدراج أبجدياً (أو تصاعدياً) حسب "المفتاح" بشكل تلقائي!

نظرة سريعة على الكود

  • #include <map>: أمر استدعاء مكتبة الخرائط لتفعيل هذه الأداة.
  • map<string, int> ages;: إنشاء خزانة اسمها ages. المفتاح فيها نص (string)، والقيمة رقم (int).
  • ages["Alice"] = 25;: افتح الدرج المكتوب عليه Alice وضع بداخله الرقم 25.
  • for (auto p : ages): حلقة تكرارية للمرور على جميع الأدراج وطباعة المفتاح والقيمة.
#include <iostream>
#include <map> // أمر استدعاء المكتبة
using namespace std;

int main() {
    map<string, int> ages; // المفتاح نص، والقيمة رقم

    ages["Alice"] = 25;
    ages["Bob"] = 30;
    ages["Charlie"] = 22;

    for (auto p : ages)
        cout << p.first << ": " << p.second << endl;
}

شاشة المخرجات (Output):

Alice: 25
Bob: 30
Charlie: 22

ملاحظة هامة: الأسماء طُبعت بالترتيب الأبجدي A ثم B ثم C، لأن الـ map يرتب المفاتيح تلقائياً.

الدرس الثاني: بناء وتجهيز الخريطة

كيف نحدد أنواع البيانات للمفتاح والقيمة.

مصنع الخزائن 🛠️

قبل استخدام الخريطة، يجب أن نخبر الكمبيوتر بوضوح: "ما هو نوع الملصق (المفتاح)؟ وما هو نوع المعلومة (القيمة)؟". الميزة الرائعة أننا أحرار في دمج أي نوعين من البيانات!

  • map<int, string>: تصلح لقاعدة بيانات الطلاب، حيث المفتاح هو "رقم الجلوس" (int) والقيمة هي "اسم الطالب" (string).
  • map<string, double>: تصلح لمتجر، حيث المفتاح "اسم المنتج" (string) والقيمة "السعر" (double - رقم عشري).
  • map<char, int>: تصلح لعد الحروف، حيث المفتاح "الحرف" (char) والقيمة "عدد مرات ظهوره" (int).
#include <map> // خطوة أساسية

// أمثلة على تشكيلات مختلفة:
map<int, string> students;     // المفتاح: رقم صحيح ، القيمة: نص
map<string, double> prices;    // المفتاح: نص ، القيمة: رقم عشري
map<char, int> frequency;      // المفتاح: حرف واحد ، القيمة: رقم صحيح

💡 نصيحة مسابقات (Pro Tip)

في المسابقات، تذكر دائماً أن الترتيب يتم بناءً على المفتاح (النوع الأول) فقط، وليس بناءً على القيمة. إذا أردت ترتيب العناصر أبجدياً اجعل النص هو المفتاح، وإذا أردتها تصاعدياً بالرقم اجعل الرقم هو المفتاح.

الدرس الثالث: الفخ الخطير! (إضافة وبحث)

طريقة الوصول للبيانات، وكيف تتجنب الأخطاء الشائعة.

الوصول السهل باستخدام []

لإضافة عنصر أو قراءته، نستخدم الأقواس المربعة ونضع داخلها "المفتاح".

  • scores["Mark"] = 90;: هذه تعليمة إضافة وتعديل في نفس الوقت. إذا كان Mark غير موجود سيتم إضافته، وإذا كان موجوداً سيتغير رقمه إلى 90.
  • cout << scores["Mark"];: لطباعة القيمة المرتبطة بـ Mark.
map<string, int> scores;

scores["Mark"] = 90;          // الإضافة باستخدام الأقواس []
cout << scores["Mark"];       // يطبع 90

⚠️ الفخ: ماذا لو بحثنا عن مفتاح غير موجود؟

تخيل أنك تبحث عن نتيجة طالب اسمه "Unknown" وهو غير مسجل في النظام. في لغات برمجة أخرى سيحدث خطأ (Error)، لكن لغة C++ تتصرف بشكل غريب جداً!

  • عندما نستخدم الأقواس المربعة للبحث عن مفتاح غير موجود، فإن البرنامج يُنشئ هذا المفتاح فوراً ويعطيه قيمة صفرية (0). هذا قد يخرب بياناتنا في المسابقات!
cout << scores["Unknown"];  // المفاجأة: لن يحدث خطأ، بل سيطبع 0 وينشئ درجاً جديداً له!

الحل الآمن: استخدام الرادار count()

  • دالة count("Tom") تتحقق بهدوء: هل Tom موجود؟ (تعيد 1 إذا وجدته، و 0 إذا لم تجده) دون أن تنشئ درجاً جديداً له.
// قبل الطباعة، تأكد من وجوده باستخدام الرادار
if (scores.count("Tom")) {
    cout << "Tom found: " << scores["Tom"];
}

الدرس الرابع: لوحة التحكم (العمليات الشائعة)

الأدوات الأساسية لإدارة محتويات الخريطة.

العملية (الوظيفة) المثال بالكود الشرح المبسط
الإضافة والوصول [] m["Bob"] = 10; لإضافة عنصر جديد أو قراءة وتعديل قيمة موجودة.
الحذف erase() m.erase("Bob"); يحذف "الدرج" كاملاً (المفتاح والقيمة معاً).
التحقق count() if (m.count("Bob")) يعيد 1 إذا كان المفتاح موجوداً، و 0 إذا لم يكن.
العدد size() m.size(); يخبرنا بعدد الأزواج (العناصر) الموجودة حالياً.
التدمير clear() m.clear(); يفرغ الخزانة تماماً ويحذف جميع العناصر.
كاشف الفراغ empty() if (m.empty()) يعيد True إذا كانت الخريطة فارغة.

الدرس الخامس: استخراج البيانات (Iteration)

كيف نمر على كل محتويات الخريطة لطباعتها.

عندما نستخدم حلقة التكرار للمرور على عناصر الـ Map، نحن لا نستخرج قيمة واحدة، بل نستخرج "زوجاً" (Pair) في كل خطوة (المفتاح والقيمة معاً).

الطريقة الحديثة: القطار السريع 🚆

  • auto p: الزوج المستخرج الحالي.
  • p.first: الكلمة السرية لاستخراج "المفتاح" (الاسم).
  • p.second: الكلمة السرية لاستخراج "القيمة" (العمر).
for (auto p : myMap) {
    // p.first  يعني المفتاح
    // p.second يعني القيمة
    cout << p.first << " -> " << p.second << endl;
}

طريقة المحترفين: المفتش (Iterator) 🕵️

  • في طريقة المؤشر، نستخدم السهم -> بدلاً من النقطة للوصول إلى first و second.
  • هذه الطريقة تستخدم في المسابقات عند الحاجة للبحث عن نقطة معينة والبدء منها بدلاً من المرور على الخريطة كاملة.
for (auto it = myMap.begin(); it != myMap.end(); it++) {
    // لاحظ استخدام علامة السهم
    cout << it->first << ": " << it->second << endl;
}

الدرس السادس: تحدي عداد الكلمات

تطبيق عملي: أشهر مسألة في البرمجة التنافسية باستخدام الخرائط.

التحدي: إعطاؤك جملة طويلة، والمطلوب حساب كم مرة تكررت كل كلمة فيها! باستخدام الـ Map، يمكننا حل هذه المسألة الصعبة بأسطر قليلة وبطريقة عبقرية.

  • map<string, int> freq;: خريطة مفتاحها الكلمة، وقيمتها عدد مرات الظهور.
  • تقوم الحلقة بتجميع الحروف لتكوين الكلمة. عندما تجد مسافة، تعتبر الكلمة مكتملة.
  • freq[word]++: السطر السحري! يذهب لدرج الكلمة، وإذا لم تكن موجودة ينشئها بقيمة 0 ثم يضيف 1 فتصبح 1. وإذا ظهرت مجدداً يرفع العدد إلى 2... وهكذا.
#include <iostream>
#include <map>
#include <string>
using namespace std;

int main() {
    string text = "apple banana apple orange banana apple";
    map<string, int> freq; // الخريطة السحرية

    string word;
    for (auto c : text) {
        if (c == ' ') { // إذا وجدنا مسافة، انتهت الكلمة
            if (!word.empty()) freq[word]++; // زد العداد بمقدار 1
            word.clear(); // فرغ المتغير لنبدأ كلمة جديدة
        } else {
            word += c; // تجميع الحروف لتكوين كلمة
        }
    }
    if (!word.empty()) freq[word]++; // لتسجيل آخر كلمة في الجملة

    // طباعة التقرير النهائي
    for (auto p : freq)
        cout << p.first << ": " << p.second << endl;
}

المخرجات (ستظهر مرتبة أبجدياً!):

apple: 3
banana: 2
orange: 1

الدرس السابع: عائلة الـ Maps (السلاح السري)

الفرق بين الخريطة العادية والخريطة السريعة جداً.

الخريطة العادية map

  • ✅ مرتبة: ترتب المفاتيح دائماً.
  • ❌ المكرر: لا تسمح بتكرار المفاتيح.
  • ⏱️ السرعة: O(log n) (ممتازة).
  • طريقة العمل: شجرة متوازنة.

الخريطة الفوضوية unordered_map

  • ❌ مرتبة: عشوائية تماماً.
  • ❌ المكرر: لا تسمح بتكرار المفاتيح.
  • 🚀 السرعة: O(1) متوسطاً (خارقة!).
  • طريقة العمل: جداول التجزئة (Hashing).

مثال تطبيقي: الخريطة الخارقة

  • #include <unordered_map>: أمر استدعاء خاص بها.
  • نفس الأوامر السابقة تماماً، الفرق الوحيد أن هذه الخريطة لا تهتم بترتيب الأسماء، لكنها تعطيك النتيجة بسرعة الضوء لتنقذك في المسابقات!
#include <iostream>
#include <unordered_map> // أمر استدعاء مختلف
using namespace std;

int main() {
    // تعريف الخريطة السريعة
    unordered_map<string, int> age;

    age["Bob"] = 30;
    age["Alice"] = 25;
    age["Tom"] = 28;

    for (auto p : age)
        cout << p.first << ": " << p.second << endl;
}

الناتج سيكون عشوائياً (مثلاً قد يطبع Tom ثم Bob ثم Alice). استخدمها عندما يكون "الترتيب" غير مطلوب في السؤال، وتكون "السرعة القصوى" هي هدفك لتجنب (Time Limit).
